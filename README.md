# RF Rust

A Rust implementation of Random Forest algorithms.

## Description

This project provides a clean and efficient implementation of Random Forest machine learning algorithms in Rust.

## Features

- Fast Random Forest implementation
- Terminal User Interface (TUI) for interaction
- Built with performance and safety in mind

## Prerequisites

- Rust (latest stable version)
- Cargo package manager

## Installation

Clone the repository and build:

```bash
git clone https://github.com/Chihab-02/rf_rust.git
cd rf_rust
cargo build --release
```

## Usage

Run the application:

```bash
cargo run
```

## Project Structure

- `src/main.rs` - Main entry point
- `src/tui.rs` - Terminal user interface implementation
